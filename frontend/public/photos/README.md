# 📁 Фотографии базы отдыха «Олтуш»

Положите свои фото в соответствующие папки. При сборке фронтенда (`npm run build`) они автоматически попадут в production.

## Структура

```
photos/
├── items/          ← Фото объектов (домики, баня, купель)
│   ├── 01_domik1/
│   ├── 02_domik2/
│   ├── ...
│   ├── 09_banya/
│   └── 10_kupel/
└── base/           ← Фото общих зон (территория, река, лес)
```

## Правила именования

| Что | Имя файла | Пример |
|-----|-----------|--------|
| Главное фото объекта | `{id}.jpg` | `1.jpg` |
| Галерея объекта | `{id}_1.jpg`, `{id}_2.jpg` … | `1_1.jpg`, `1_2.jpg` |
| Фото общих зон | `base/любое_имя.jpg` | `base/territory.jpg` |

**Поддерживаемые форматы:** `.jpg`, `.jpeg`, `.png`, `.webp`

**Рекомендации:**
- Размер: 800×600 px или крупнее
- Вес: до 300 КБ на фото
- Соотношение сторон: 4:3 для галереи, 1:1 для stories/аватарок

Если фото для объекта не найдено — покажутся красивые placeholder-фото с Unsplash.
