# Фото общих зон базы отдыха

Сюда можно положить фото территории, реки, леса, общей инфраструктуры.

## Примеры имён

| Файл | Что это |
|------|---------|
| `territory.jpg` | Общий вид территории |
| `river.jpg` | Река / водоём |
| `forest.jpg` | Лес |
| `entrance.jpg` | Въезд на базу |
| `playground.jpg` | Детская площадка |
| `parking.jpg` | Парковка |

> Эти фото не привязаны к объектам и используются по прямой ссылке в коде.
